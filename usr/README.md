# Web Landing Page App

Proyek ini adalah contoh aplikasi web berupa **Landing Page** yang dikembangkan menggunakan Flutter. Desainnya dibuat sangat responsif sehingga tampilannya dapat menyesuaikan otomatis di perangkat Desktop maupun Mobile (HP).

## Fitur Utama
- **Desain Responsif**: Secara mulus beradaptasi antara layar lebar (Desktop) dan layar kecil (Mobile).
- **Bagian Hero**: Memuat penawaran utama dan tombol "Call to Action" (Mulai).
- **Bagian Fitur**: Kartu fitur-fitur keunggulan dalam grid/wrap otomatis.
- **Header Tetap (Pinned Header)**: Navigasi yang terus muncul meskipun pengguna menggulir ke bawah.
- **Seksi Call To Action (Ajakan)**: Baris aksi untuk konversi pengguna.
- **Footer**: Hak cipta dan informasi merek.

## Tech Stack
- **Framework**: Flutter
- **Platform Utama**: Web, Desktop, dan iOS/Android
- **Bahasa**: Dart

## Cara Menjalankan
1. Pastikan Anda sudah menginstal Flutter versi terbaru.
2. Clone atau unduh repositori ini.
3. Jalankan `flutter pub get` untuk memastikan semua dependensi siap.
4. Jalankan perintah `flutter run -d chrome` untuk menjalankannya di peramban web sebagai halaman web penuh.
